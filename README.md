# tw-API

## ./.env
```text
TW_KEY=<consumer key>
TW_KEY_SEC=<consumer key_secret>
TW_ACC_TOKEN=<access token>
TW_ACC_TOKEN_SEC=<access token_secret>
TW_BEARER_TOKEN=<bearer token>
```

## Usage

### Twitter API v1.1
```python
api = generate_api_v1()
```

### Twitter API v2
```python
client = generate_api_v2()
```
